# Сложно сосредоточиться

## Описание

Адаптивная верстка страницы с поддержкой нескольких цветовых тем и разрешений экрана.
Подход **Mobile First**.

## Этапы реализации

1. Верстка основного лейаута для темной темы и разрешения 375 x 668
2. Медиазапрос для разрешения 768 x 1024
3. Медиазапрос для разрешения 1024 x 768
4. Светлая тема. Отладка переключения тем и разрешений
