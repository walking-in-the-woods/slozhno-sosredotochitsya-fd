# Сложно сосредоточиться

## Описание

Адаптивная верстка страницы с поддержкой нескольких цветовых тем и разрешений экрана.
Подход **Mobile First**.

## Этапы реализации

1. Верстка основного лейаута для темной темы и разрешения 375 x 668
2. Медиазапрос для разрешения 768 x 1024
3. Медиазапрос для разрешения 1024 x 768
4. Светлая тема. Отладка переключения тем и разрешений
5. Коррекция по замечаниям ревьюера:
  - Удалены дубликаты свойств в медиазапросах;
  - Фоновое изображение привязано к экрану;
  - Откорректированы цвета кнопок, ссылок и декоративных элементов;
  - Добавлены значения по умолчанию для переменных;
  - Галерея оформлена в виде ненумерованного списка;
